# Udainthorai Uruvakkuvom Charitable Trust Static Website

This is a static multi-page website for Udainthorai Uruvakkuvom Charitable Trust.

## Pages

- `index.html` - Home page with hero, objective block, impact counters, and program pillars.
- `about.html` - Vision, mission, founder profile, official logo, and embedded bio presentation.
- `gallery.html` - Tabbed healthcare, education, and community development galleries.
- `transparency.html` - Trust verification documents, certificate previews, CSR record, and awards section.
- `donate.html` - Finance breakdown, bank transfer table, and UPI QR payment details.
- `contact.html` - Contact cards, form UI, and embedded map.

## Deploying on Render

Create a new Static Site on Render and point it to this repository. The included `render.yaml` uses:

- `staticPublishPath: .`
- no build command
- static publishing from the repository root

Before launch, connect the contact form to an email, CRM, or form service if live submissions are required.
